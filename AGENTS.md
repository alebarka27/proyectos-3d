# Proyecto: proyects-3d

## Stack
- Node.js (API server)
- Frontend vanilla HTML/CSS/JS en `/public`
- MercadoLibre API (OAuth2, webhooks)
- Desplegado en Vercel

## Estado actual
- API funcional con rutas para productos y MercadoLibre
- Frontend con catálogo, importación desde ML, y página de producto
- Integración con MercadoLibre: autenticación OAuth2, webhooks, importación de productos
